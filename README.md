## Setup
```
$ make install
$ brain-games
```
## Brain even
Answer "yes" if the number is even, otherwise answer "no".
```
$ brain-even
```
[How to install and play](https://asciinema.org/a/edtydMZvyBDnNW1MBwbuCnKFe)
## Brain calc

